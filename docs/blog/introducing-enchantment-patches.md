# New Defaulted Features Coming Straight from YOUR House!

So to preface, in this new Defaulted update I have added [enchantment patches](../defaulted/enchantment-intro) which are a way to modify enchantments without outright replacing them.

So uh, quite frankly, I was thinking about this for a while, but isn't it kind of dumb that there's no way to edit enchantments without overwriting them?

Like, I don't know how no one thought of this before, it wasn't even that hard to make.

I hope that you all find this as cool as I do, not much else to say, heh.
